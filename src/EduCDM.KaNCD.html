class EduCDM.KaNCD.KaNCD.KaNCD(meta_data: dict, dim: int = 40, mf_type: str = 'gmf', layer_dim1: int = 512, layer_dim2: int = 256)

Bases: CDM

The KaNCD model. :param meta_data: a dictionary containing all the userIds, itemIds, and skills. :param dim: the dimension of the latent vectors of users, items and skills. default: 40 :param mf_type: the type of layer(s) to be used for the interaction among latent vectors. default: “gmf”

  • “mf”: matrix factorization, just inner product

  • “gmf”: general matrix factorization. This is the layer used in the paper.

  • “ncf1”: neural collaborative filtering, one layer.

  • “ncf2”: neural collaborative filtering, two layers.

Parameters:

  • layer_dim1 – the dimension of the first hidden layer. Default: 512

  • layer_dim2 – the dimension of the second hidden layer. Default: 256

Examples::

meta_data = {‘userId’: [‘001’, ‘002’, ‘003’], ‘itemId’: [‘adf’, ‘w5’], ‘skill’: [‘skill1’, ‘skill2’, ‘skill3’, ‘skill4’]} model = KaNCD(meta_data, 40, ‘gmf’, 512, 256)

save(filepath)

Save the model. This method is implemented based on the PyTorch’s torch.save() method. Only the parameters in self.ncdm_net will be saved. You can save the whole NCDM object using pickle.
